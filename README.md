
# SKHU-PS

> 성공회대 학생들의 프로젝트 진행을 보다 수월하게 하는 기능을 제공하는 홈페이지   
> 프로젝트 인원 모집+ToDoList+교수 권한+타임라인+게시판+경진대회 조회

### 기술
* Spring boot
* JPA
* Axios
* Vue.js
* Bootstrap
* Google SMTP
* MySQL
